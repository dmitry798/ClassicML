
ClassicML — компактная библиотека классического машинного обучения на C++ с нуля. Библиотека предоставляет матричную алгебру, препроцессинг данных, оптимизаторы, модели (линейная/логистическая регрессия, KNN‑классификация/регрессия, K‑Means), а также метрики качества и удобный пайплайн обучения/инференса.

## Ключевые особенности

- Нативный C++ без внешних зависимостей. Собственная матричная библиотека, препроцессинг и оптимизация.
    
- Полный цикл ML: подготовка данных → обучение → предсказание → оценка метрик.
    
- Модели:
    
    - Линейная регрессия: градиентные методы и точное решение через SVD.
        
    - Логистическая регрессия: бинарная и мультиклассовая (softmax).
        
    - KNN классификатор/регрессор: uniform и distance weighting, L1/L2 расстояния.
        
    - K‑Means: инициализация “base” и k‑means++ и выбор метрики (евклид/манхэттен).
        
- Препроцессинг: StandardScaler, One‑Hot Encoder, train/test split, нормализация в инференсе cнаружи через сохранённые статистики.
    
- Метрики: MSE, RMSE, MAE, R², logloss/softmax‑logloss, accuracy, precision, recall, F1, inertia.
    

## Для кого

- Студенты и разработчики, кому важна прозрачная реализация алгоритмов без «магии» сторонних фреймворков.
    
- Авторы учебных и исследовательских проектов, которым нужен контролируемый, читаемый C++‑код.
    

## Структура проекта

- include/ClassicML/
    
    - matrix.h, matrxop.h: собственная матричная алгебра (хранение, слайсы строк/столбцов, сортировки, unique(), трансформации, transpose и др.).matrix.h​
        
    - preprocessor.h: StandardScaler, OneHotEncoder, split.
        
    - models.h: базовый контейнер данных и интерфейсы моделей/тренера/дистанций.LogisticRegression.cpp​
        
    - optimization.h: API оптимизаторов (GD, SGD, Momentum, Nesterov, SVD).
        
    - errors.h: метрики и отчёты качества.
        
    - ClassicML.h: единая точка подключения заголовков.ClassicML.h​
        
- src/
    
    - matrix.cpp: реализация матричных операций.KnnClassifier.cpp​
        
    - preprocessor.cpp: scaler, one‑hot, сплит и нормализация/денормализация.
        
    - optimization.cpp: GD, SGD, Momentum, Nesterov, SVD тренинг.
        
    - models.cpp: общий Trainer (выбор оптимизатора), единый predict c корректной нормализацией.models.cpp​
        
    - LinearRegression.cpp, LogisticRegression.cpp, KnnClassifier.cpp, KnnRegression.cpp, KMeans.cpp: реализации моделей.errors.h+3​
        
    - errors.cpp: метрики и печать отчётов.errors.cpp​
        
    - main.cpp: примеры использования (Titanic и синтетика).main.cpp​
        

## Сборка

- Требования: C++14+ и любой современный компилятор (g++/clang++/MSVC).
        
- Быстрый старт:

    - Библиотека готова к использованию!
